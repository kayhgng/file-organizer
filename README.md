# 📂 File Organizer — مرتب‌ساز فایل

## ✨ Features

- 🖥️ **Beautiful dark UI** — clean and minimal interface built with Tkinter
- 📊 **Live preview** — see how many files are in each category before sorting
- ⚡ **One click** — pick a folder, hit the button, done
- 🔁 **No overwrites** — duplicate filenames are renamed automatically
- 📋 **Real-time log** — watch every file move as it happens
- 🪶 **Zero dependencies** — uses only Python's standard library

## 🚀 Getting Started

**No installation needed.** Just run:

```bash
git clone https://github.com/kayhgng/file-organizer
cd file-organizer
python file_organizer.py
```

> Requires Python 3.8 or higher. That's it.

---

## 🎮 How to Use

1. Run the script
2. Click **"انتخاب فولدر"** to pick your messy folder
3. Check the category cards to see the file preview
4. Hit **▶ شروع مرتب‌سازی** and watch the magic happen

---

## 🛠️ Built With

- **Python 3** — core logic & file operations
- **Tkinter** — cross-platform GUI
- **threading** — non-blocking UI during sorting
- **shutil / pathlib** — safe file moving
